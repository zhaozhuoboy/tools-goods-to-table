# 图标文件说明

请在此目录下放置以下图标文件：

- `icon16.png` - 16x16 像素
- `icon48.png` - 48x48 像素  
- `icon128.png` - 128x128 像素

## 图标设计建议

- 主色调：京东红 (#E1251B)
- 可以使用简单的表格或同步图标
- 确保在不同尺寸下清晰可见

## 生成图标

可以使用以下工具生成图标：
- [Favicon Generator](https://www.favicon-generator.org/)
- [Icon Generator](https://icon.kitchen/)
- 或使用设计工具（Figma、Sketch 等）导出
